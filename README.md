
CdiCalendar
=======

**Que es CdiCalendar**

CdiCalendar es un modulo para agendar recordatorios y citas


Instalacion
============

Installation via composer, agregar a composer.json:
```
"require" : {
    "cdi/cdicalendar": "dev-master"
}
```



```
'modules' => array(
    'Application',
    'CdiCalendar'             
)
```


Dependencias
============

- cdi/cdicommons
- cdi/cdiuser
```
"require" : {
    "cdi/cdicommons": "dev-master",
    "cdi/cdiuser": "dev-master"
}
```
